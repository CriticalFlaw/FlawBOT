Commands for miscellaneous bot interactions.

### ask
Asks the magic 8-ball a question.
```
/ask Am I lucky?
```

### cat
Returns a random cat fact and image.
```
/cat
```

### coinflip
Flips a coin.
```
/coinflip
```

### diceroll
Rolls a six-sided die.
```
/diceroll
```

### dog
Returns a random dog image.
```
/dog
```
