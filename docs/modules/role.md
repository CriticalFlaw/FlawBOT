Commands for managing server roles.

### create
Creates a new server role.
```
/role create admin
```

### delete
Deletes a server role.
```
/role delete admin
```

### info
Returns server role information.
```
/role info admin
```

### inrole
Returns list of users in a given role.
```
/role inrole admin
```

### mention
Toggles whether the role can be mentioned by others.
```
/role mention admin
```
